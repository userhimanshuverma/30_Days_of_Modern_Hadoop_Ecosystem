# 🚀 30 Days of Modern Hadoop Ecosystem
**Tagline:** From HDFS to Production Data Platforms

A world-class open-source curriculum designed for data engineers, platform architects, and SREs seeking to master the storage, processing, messaging, OLAP, and security components of modern distributed data infrastructure.

## 📅 Curriculum Overview
* **Phase 1: Foundations & Storage (Days 1–8)** - HDFS, YARN, HA, and CGroups
* **Phase 2: Ingestion & Messaging (Days 9–13)** - ZooKeeper, Kafka, Schema Registry, and Connect
* **Phase 3: Execution Engines (Days 14–19)** - MapReduce, Tez, Spark Core, SQL, Streaming, and Tuning
* **Phase 4: Structured Storage & OLAP (Days 20–24)** - Hive, LLAP, ACID, Trino MPP, and Pinot OLAP
* **Phase 5: Governance, Security & Orchestration (Days 25–30)** - Knox, Ranger, Atlas, Airflow, and Observability

## 🏗️ Repository Layout
* `/hands-on-labs` - Docker Compose stacks, datasets, and integration tests
* `/build-from-source` - Compilation and packaging playbooks for all core tools
* `/templates` - Reusable markdown templates for lessons, builds, and case studies
* `/case-studies` - SRE post-mortems and architecture migrations from Netflix, Uber, etc.
